# Fix participants & results

Correcting a tournament after the fact.

1. **Open the tournament.** Admin → **Tournaments** → select the event.
2. **Edit a record.** Change a player's **W**, **L** or **D** in the participants
   table. The change saves automatically when the field loses focus; **Pts**
   updates with it.
3. **Reassign a row.** If a result was logged against the wrong person, use the
   player picker on that row to point it at the right player. The match record
   moves with it.
4. **Add a missing player.** Use the bottom **add** row: pick the player, enter
   W/L/D, then **add**.
5. **Remove a row.** Click **remove** on the row (you'll be asked to confirm).

**Sorting (view only).** Click the **Player** or **Pts** header to sort the table
while you check results. This never changes what gets saved or the stored order —
it's purely to help you read the table.
