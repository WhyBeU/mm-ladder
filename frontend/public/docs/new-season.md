# Start a new season + link a cup

Setting up the next block of Monday nights.

1. **Create the season.** Admin → **Seasons** → add a season.
2. **Set the basics:**
   - **Set code** — the MTG set being drafted (e.g. `BLB`). Drives the season's
     set symbol and the "champion" icon.
   - **Start / end dates** — the season window. The season counts as *current*
     while today falls inside it.
   - **Event count** — how many Monday nights the season runs.
   - **Qualifying type** — `POINTS` (rank by total points) or `BEST` (rank by the
     total of each player's top-N event scores).
3. **Link the yearly cup.** Set the season's **yearly cup** so its results feed the
   right cup. Once linked, the cup-qualifier line appears on the season ladder.
4. **Run it.** New tournaments created under this season now feed its standings.
   See *Run a Monday night*.

> The newest current season is what visitors land on by default.
