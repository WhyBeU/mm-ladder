# Champions, POTY & cup winners

Marking the honours that show up as icons on the ladder, podium and cup view.

## Season champion

- Admin → **Seasons** → open the season → set the **champion** (the winning player).
- This shows the set-symbol award icon next to the player across the app.

## Player of the Year (POTY)

- Admin → **Cups** → open the yearly cup → set **Player of the Year**.
- POTY is a yearly honour, so it lives on the cup, not a single season.

## MM Cup winner

- Admin → **Cups** → open the yearly cup → set the **cup champion**.
- Marks that player as the winner of that year's cup.

## Qualified players (for a cup)

- Admin → **Cups** → open the cup → edit the **qualified players** list.
- Qualified players get a gold check on the season ladder and appear on the cup's
  **Qualified players** cards. Qualification is set manually here — it isn't
  auto-computed.

> All of these are display/award fields — they don't change points or standings.
