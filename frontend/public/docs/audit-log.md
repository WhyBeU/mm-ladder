# Read the audit log

Every admin change is recorded so you can see what happened and when.

- Admin → **History** tab.
- Each entry shows **what** was changed, **which** record, and **when**.
- Use it to confirm a correction landed, or to retrace a surprising change to the
  standings (e.g. a participant edit or a player merge).

> The audit log is read-only — it's a record of actions, not a place to undo them.
> To reverse something, make the opposite change in the relevant section.
