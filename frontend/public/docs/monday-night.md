# Run a Monday night

The weekly flow from creating the event to seeing the ladder update.

1. **Create the tournament.** Admin → **Tournaments** → add a tournament under the
   live season. Set its date and (optionally) a name.
2. **Add participants.** Open the tournament and add each player with their match
   record (**W / L / D**). Points are computed automatically: **3 per win, 1 per
   draw**.
3. **Fix mistakes anytime.** Edit any W/L/D field (it saves when you click away),
   reassign a row to a different player, or remove a row. See
   *Fix participants & results*.
4. **What recomputes.** Season standings, points, **trophies** (a clean 3-0 = 9
   points), averages and the **Best** total all recompute as soon as results are
   saved. The public ladder reflects the new tournament immediately — no publish step.

> Tip: sort the participants table by **Pts** to sanity-check the night's results.
